Wikipedia Policy Violations

GenuVerity's methodology explicitly prohibits Wikipedia as a source because it is user-edited and not a primary source. We found 32 Wikipedia links across 8 reports:

  • doppelganger-analysis - 15 Wikipedia links
  • network-analysis - 6 Wikipedia links
  • disinformation-architecture-2025 - 5 Wikipedia links
  • trump-admin-financial-conflicts - 3 Wikipedia links
  • epstein-air-logistics - 1 Wikipedia link
  • grok-ai-misinformation-crisis - 1 Wikipedia link
  • plaque-fact-check - 1 Wikipedia link

Remediation Plan: Each Wikipedia link will be replaced with the primary source that Wikipedia itself cites, or with an equivalent authoritative source.

How This Audit Was Conducted

This audit was performed using an automated tool (tools/audit-reports.js) that scans all 182 HTML report files and cross-references them with the registry (js/reports-data.js).

Metrics Collected

  • Source Counting: Parsed HTML for id="source-N" patterns in the sources sidebar
  • Source Classification: Categorized URLs by domain (.gov, .edu, major news outlets, other)
  • Wikipedia Detection: Searched all href attributes for wikipedia.org links
  • Color Compliance: Scanned for forbidden hex codes (#8b5cf6 and related purples)
  • Chart Detection: Checked for Chart.js instantiation (new Chart()
  • Verdict/Category Distribution: Aggregated from reports-data.js registry

Scoring Formula

The overall score is a weighted average:

Overall = (Sources × 35%) + (Methodology × 25%) + (Technical × 25%) + (Transparency × 15%)

Each component starts at 100 and receives penalties for violations and bonuses for exceeding standards.
