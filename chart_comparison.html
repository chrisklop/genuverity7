Visual Comparison: Fact Check Metrics

Current (DIVs)

Cons: Dependent on browser rendering (flexbox), disjointed if zoomed, heavier DOM.

Proposed (SVG)

Pros: Pixel-perfect scaling, crisp edges, part of the DOM but lightweight.

Library (Chart.js)

Cons: Requires external library (100kb+), canvas text can be blurry, overkill for simple previews.
